DOM API 即是修改文档结构的 API，通过 DOM API 可以方便的在文档中新增元素、注释或是文本节点，也能够将一小段 HTML 片段插入文档，或是在在文档中移除这些元素。Jumony 提供了标准统一的DOM API用于修改文档结构，对于所有支持修改的HTML DOM，这些API提供了一致的行为。

添加节点

添加节点的 API 均以 Add 开头，可以在 DOM 中自由添加任意节点。

IHtmlElement IHtmlContainer.AddElement( [int index], string elementName )
说明：

在容器中添加一个元素

参数:

elementName: 元素名

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

返回值：

返回添加在DOM中的元素

IHtmlTextNode IHtmlContainer.AddTextNode( [int index], string htmlText )
说明：

在容器中添加一个文本节点

参数:

htmlText: HTML文本

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

返回值：

返回添加在DOM中的文本节点

IHtmlComment IHtmlContainer.AddComment( [int index], string comment )
说明：

在容器中添加一个注释节点

参数:

comment: 注释内容

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

返回值：

返回添加在DOM中的注释节点

IEnumerable<IHtmlNode> IHtmlContainer.AddFragment( [int index], IHtmlFragment fragment )
说明：

在容器中添加一个文档片段

参数:

fragment: 要添加的文档碎片

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

返回值：

返回添加在DOM中的节点列表

IEnumerable<IHtmlNode> IHtmlContainer.AddFragment( [int index], string html )
说明：

在容器中添加一个文档片段

参数:

html: 要添加的文档片段 HTML ，将会被自动解析为 IHtmlFragment 然后被添加。

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

返回值：

返回添加在DOM中的节点列表

IHtmlElement IHtmlContainer.AddCopy( [int index], IHtmlElement element )
说明：

在容器中添加元素的拷贝

参数:

element: 要添加拷贝的元素。

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

返回值：

返回添加在DOM中的元素

IHtmlTextNode IHtmlContainer.AddCopy( [int index], IHtmlTextNode textNode )
说明：

在容器中添加文本节点的拷贝

参数:

textNode: 要添加拷贝的文本节点。

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

返回值：

返回添加在 DOM 中的文本节点

IHtmlTextNode IHtmlContainer.AddCopy( [int index], IHtmlComment comment )
说明：

在容器中添加注释节点的拷贝

参数:

comment: 要添加拷贝的注释节点。

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

返回值：

返回添加在 DOM 中的注释节点

IHtmlTextNode IHtmlContainer.AddCopy( [int index], IEnumerable<IHtmlNode> nodes )
说明：

在容器中添加一批节点的拷贝

参数:

nodes: 要添加拷贝的节点列表。

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

返回值：

返回添加在 DOM 中的节点列表

移除节点

移除节点的 API 主要包括 ClearNodes 和 Remove ，前者移除容器所有子节点，后者则是移除节点自身

IHtmlContainer IHtmlContainer.ClearNodes()
说明：

移除容器所有的子节点

返回值：

返回容器自身便于链式调用

void IHtmlNode.Remove()
说明：

从 DOM 中移除这个节点

void IEnumerable<IHtmlNode>.Remove()
说明：

从 DOM 中移除这一批节点

void IHtmlAttribute.Remove()
说明：

从 DOM 中移除这个属性

void IEnumerable<IHtmlAttribute>.Remove()
说明：

从 DOM 中移除这一批属性

替换节点

替换节点的 API 可以将节点用指定的 HTML 片段替换。

IEnumerable<IHtmlNode> ReplaceWith( IHtmlFragment fragment )
说明：

将节点用指定的文档片段替换

参数:

fragment: 用于替换节点的文档片段。

返回值：

返回替换在 DOM 中的节点列表

IEnumerable<IHtmlNode> ReplaceWith( string html )
说明：

将节点用指定的文档片段替换

参数:

html: 用于替换节点的的文档片段 HTML ，将会被自动解析为 IHtmlFragment 然后替换。

返回值：

返回替换在 DOM 中的节点列表

创建副本

创建副本的 API 可以创建 HTML 片段或文档的副本

IHtmlFragment IHtmlFragment.MakeCopy()
说明：

为文档片段创建副本

返回值：

文档片段的副本

IHtmlDocument IHtmlDomProvider.MakeCopy( IHtmlDocument document )
说明：

创建指定文档的副本

参数:

document: 要创建副本的文档。

返回值：

指定文档的副本。

文档片段相关

文档片段相关的 API 主要用来创建 IHtmlFragment 即文档片段对象

IHtmlFragment IHtmlDocument.CreateFragment()
说明：

创建文档片段。

返回值：

创建的文档片段。

IHtmlFragment IHtmlDocument.ParseFragment( string html )
说明：

解析并创建文档片段.

参数:

html: 要解析的文档片段 HTML 文本。

返回值：

创建的文档片段。

内容相关

内容相关的 API 包括 InnerHtml 和 InnerText ，可以比较直观的达到设置 InnerHtml 和 InnerText 的效果

IHtmlElement IHtmlElement.InnerHtml( string html )
说明：

设置元素的 InnerHTML 。

参数:

html: 用于替换元素内容的 HTML 文本。

返回值：

元素自身，便于链式调用。

IHtmlElement IHtmlElement.InnerText( string text, [bool encodeWhiteSpaces] )
说明：

设置元素的 InnerText 。

参数:

text: 用于替换元素内容的纯文本。

encodeWhiteSpaces: 指定是否要对连续的空白字符编码，默认是 false ，即标准的 InnerText 行为，并不会对连续的空白字符编码

返回值：

元素自身，便于链式调用。
